# Countdown Timer

A simple countdown timer web application built with HTML, CSS, and JavaScript.

## Features

- Clean and user-friendly interface
- Easy-to-use countdown timer
- Real-time updates
- Responsive design

## Files

- `index.html` - Main HTML structure
- `script.js` - JavaScript functionality
- `style.css` - Styling and layout

## How to Use

1. Open `index.html` in your web browser
2. Set your desired countdown time
3. Click start to begin the timer
4. Watch as the countdown progresses

## Installation

Simply clone the repository and open `index.html` in your browser:

```bash
git clone https://github.com/MohammadAfzal101/count-down.git
cd count-down
```

Then open `index.html` in your preferred web browser.
